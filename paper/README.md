# Paper

Full research paper: *Parametric Physics-Informed Neural Networks for European Option Pricing: A Comparative Study on AAPL Market Data*

See `paper.pdf` for the compiled version.

## Build from source

```bash
make
```

Or open `paper.tex` in Overleaf (upload all `.tex`, `.sty`, `.bst`, `.bib`, and image files from this directory).

## Files

- `paper.tex` / `paper.pdf` — main paper (ICML 2014 template)
- `references.bib` — bibliography
- `fig1_surface_comparison.png` — PINN vs BSM vs market prices across three tenors
- `fig2_error_by_tenor.png` — absolute pricing error by tenor
- `fig3_vol_smile.png` — market implied volatility vs constant σ
- `fig4_delta_surface.png` — PINN-derived Delta surface via autograd
- `table1_results.csv` — MAE and relative error by tenor (PINN vs BSM)
- `table2_moneyness.csv` — MAE breakdown by moneyness (ITM / ATM / OTM)

Figures and tables are generated by running `code/PINN_code.ipynb`.
